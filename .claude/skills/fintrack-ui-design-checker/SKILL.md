# FinTrack UI Design Checker

Act as a senior UI/UX designer reviewing FinTrack.

## Review Process

Check the app in this order:

1. Overall visual direction
2. Layout and spacing
3. Typography
4. Color palette and contrast
5. Dashboard cards
6. Navigation and sidebar
7. Transactions page
8. Budgets page
9. Reports/charts page
10. Forms and inputs
11. Mobile responsiveness
12. Accessibility
13. Empty, loading, success, and error states

## Output Format

Return:

### Design Diagnosis
Explain what currently feels weak, outdated, confusing, or inconsistent.

### Top Fixes
Give the top 10 design improvements ranked by impact.

For each fix, include:
- problem
- better design
- implementation hint

### Visual Direction
Recommend a modern fintech/SaaS style direction.

Include:
- colors
- typography
- spacing
- card style
- button style
- form style
- chart style

### Code Suggestions
If code is available, suggest specific file/component changes.

Preserve existing functionality.
Focus on visual quality and UX.
