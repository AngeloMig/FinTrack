# Full UI Redesign Prompt

Review the entire FinTrack frontend design.

Important:
- FinTrack should remain a mobile-first app UI.
- Do not redesign it into a full-width desktop SaaS dashboard.
- Keep the existing phone-sized app shell unless the user explicitly asks for desktop.
- Desktop browser view should still look like a centered mobile app.

Focus on:
- modern fintech style
- dashboard hierarchy
- layout spacing
- typography
- card polish
- colors and contrast
- forms
- transaction lists
- charts
- navigation
- mobile responsiveness
- accessibility

Do not change business logic.
Do not widen the app shell or introduce desktop-only layouts.

First inspect the project structure and identify the main frontend files.

Then propose a design plan.

Then implement the highest-impact visual improvements.

After changes, run available lint/build checks.
